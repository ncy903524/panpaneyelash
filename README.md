# 美業 CRM 管理系統 ✦

專為美業工作室打造的客戶關係管理系統（第一版）

## 功能總覽

| 頁面 | 功能 |
|------|------|
| 👤 客戶資料 | 基本檔案、施作紀錄、使用優惠、照片上傳、同意書連結 |
| 🔔 提醒 | 預約提醒、生日祝福、四種服務回訪週期設定 |
| 📅 預約 | 今日行程、TimeTree/LINE 整合開關、手動新增預約 |
| 💰 成本 | 台中/台北分別統計、五種服務分類收費紀錄 |
| 📊 統計 | 收入分析、回訪率/流失預警、來源追蹤 |

## 服務項目支援

- 😴 睫毛
- ✏️ 眉毛
- 💋 霧唇
- 💜 髮際線
- 🗑 除色

## 使用方式

直接用瀏覽器開啟 `index.html`，或部署至 GitHub Pages。

### 部署至 GitHub Pages

1. 將此資料夾推送至 GitHub repository
2. 進入 Repository → Settings → Pages
3. Source 選擇 `main` branch，資料夾選 `/ (root)`
4. 儲存後等待約 1 分鐘即可透過 `https://你的帳號.github.io/repository名稱/` 瀏覽

## 技術規格

- 純 HTML / CSS / JavaScript，無需任何框架或套件
- 手機優先（Mobile First）響應式設計
- 支援 iOS PWA（可加入主畫面）
- 底部 Tab Bar 採 Pantone 4174U 色系
